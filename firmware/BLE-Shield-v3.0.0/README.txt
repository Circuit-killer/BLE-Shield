The BLE112 pins are used as follows:
--------------------------------------------------

TX                 P0_4
RX                 P0_5

Connected LED      P0_0
Interrupt Pin      P0_1

BLE Host Wakeup    P1_5
BLE Modules Wakeup P1_6

USB Port           P1_0
Boot to DFU        P1_1
--------------------------------------------------

The firmware is compiled against v 1.2.1 build 91
